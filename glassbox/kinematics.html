Rest and Motion — Kinematics

Distance vs displacement, average & instantaneous speed/velocity, and reading v–t graphs — the core ideas from Chapter 3 (HC Verma), worked examples, and an unlimited NEET-UG practice quiz.

Key ideas at a glance

Distance vs displacement

  • Distance = length of the actual path (scalar, ≥ 0).
  • Displacement = straight line from start to end (vector).
  • Semicircle radius R: distance = πR, displacement = 2R.
  • Full circle: distance = 2πR, displacement = 0.

Average speed & velocity

  • Average speed = total distance / total time.
  • Average velocity = displacement / total time.
  • For two equal-distance halves at v₁, v₂: avg speed = 2v₁v₂/(v₁+v₂).

Instantaneous speed

  • v = ds/dt (slope of the s–t graph).
  • If s = k t², then v = 2k t.

v–t graph

  • Slope of v–t graph = acceleration.
  • Area under the v–t graph = distance travelled.
  • Triangle: distance = ½ × base × height.

Uniform acceleration

  • v = u + at
  • s = ut + ½at²
  • v² = u² + 2as

Watch out

  • Average velocity can be zero while average speed is not (round trips).
  • Displacement direction points from initial → final position.

Worked examples (HC Verma Ch. 3)

3.1 — Semicircular morning walk, R = 40.0 m
Distance = length of the track = πR = π × 40.0 = 126 m.
Displacement = straight line joining the ends = diameter = 2R = 2 × 40.0 = 80 m, directed from start to finish.
3.2 — s = (2.5 m/s²)t²: average speed (0→5 s) and instantaneous speed at 5 s
(a) Distance in 5 s: s = 2.5 × (5.0)² = 62.5 m.
Average speed = 62.5 m / 5.0 s = 12.5 m/s.
(b) v = ds/dt = (2.5)(2t) = (5.0 m/s²)t. At t = 5.0 s, v = 5.0 × 5.0 = 25 m/s.
3.3 — Distance from a v–t graph (triangle: height 6 m/s, base 3 s)
Distance = area under the v–t graph = area of the right triangle = ½ × base × height = ½ × 3 s × 6 m/s = 9 m.
3.4 — Minute hand 4.0 cm: average velocity of the tip
Over 30 minutes the tip moves from the 12 to the 6 mark, so displacement = diameter = 2R = 8.0 cm.
(a) 6:00→6:30 a.m. = 1800 s: vav = 8.0 / 1800 = 4.4 × 10⁻³ cm/s.
(b) 6:00 a.m.→6:30 p.m. = 45000 s: vav = 8.0 / 45000 = 1.8 × 10⁻⁴ cm/s.
(Displacement is the same; only the time differs.)
3.5 — Uniform acceleration: u = 2.5 m/s, a = 0.50 m/s²
(a) Distance in the first 2 s — use x = ut + ½at²:
x = (2.5)(2) + ½(0.50)(2)² = 5.0 + 1.0 = 6.0 m. Since the particle never turns back, this is also the distance travelled.
(b) Time to reach v = 7.5 m/s — use v = u + at:
t = (v − u)/a = (7.5 − 2.5)/0.50 = 10 s.
(c) Distance to reach v = 7.5 m/s — use v² = u² + 2ax:
x = (v² − u²)/(2a) = (56.25 − 6.25)/(1.00) = 50 m.
3.6 — Displacement in the last 1 second (and the "tᵗʰ second" caveat)
Position at time t: s = ut + ½at². Position at (t − 1 s): s′ = u(t−1) + ½a(t−1)².
Displacement in the last second: st = s − s′ = u(1 s) + (a/2)(2t − 1 s)(1 s). …(i)
For u = 5 m/s, a = 2 m/s², t = 10 s:
st = (5)(1) + ½(2)(2×10 − 1)(1) = 5 + (1)(19)(1) = 24 m.
Correction / caveat: dropping the "(1 s)" factors gives the popular formula sₜ = u + (a/2)(2t − 1), which is dimensionally inconsistent (its terms have different dimensions). Equation (i) is the correct form. Also, this is the displacement in the last second, not necessarily the distance covered.
3.7 — Ball thrown up at 4.0 m/s: maximum height (g = 10 m/s²)
Take up as +Y, so u = +4.0 m/s and a = −g = −10 m/s². At the highest point the velocity is v = 0.
Use v² = u² + 2ay: 0 = (4.0)² + 2(−10)y.
y = 16 / 20 = 0.80 m.
(Key idea: at maximum height the velocity is momentarily zero; sign convention makes a = −g.)
3.8 — 2-D constant acceleration: a = 1.5 m/s² at 37°, u = 8.0 m/s along X
Resolve the acceleration (cos37° = 0.8, sin37° = 0.6):
aₓ = 1.5 cos37° = 1.2 m/s²; a_y = 1.5 sin37° = 0.90 m/s². Initial: uₓ = 8.0 m/s, u_y = 0.
Velocity at t = 4.0 s: vₓ = 8.0 + 1.2×4 = 12.8 m/s; v_y = 0 + 0.90×4 = 3.6 m/s.
Speed v = √(12.8² + 3.6²) = 13.3 m/s, at tanθ = v_y/vₓ = 3.6/12.8 = 9/32 (θ ≈ 15.7° above X).
Position at t = 4.0 s: x = 8.0×4 + ½×1.2×4² = 41.6 m; y = 0 + ½×0.90×4² = 7.2 m.
So the particle is at (41.6 m, 7.2 m).
3.9 — Range of a projectile: 12.0 m/s at 45° (g = 10 m/s²)
Horizontal range R = u²sin2θ/g = (12)²·sin(2×45°)/10.
Since sin90° = 1: R = 144 × 1 / 10 = 14.4 m.
(Pro-tip: 45° gives sin2θ = 1, so the range is maximum at this angle.)
3.10 — River swimmer: 4.0 km/h across a 3.0 km/h current
Swimmer aims perpendicular to the flow, so the two velocities are at 90°.
vS,G = √(vS,R² + vR,G²) = √(4.0² + 3.0²) = 5.0 km/h (a 3-4-5 triangle).
Angle with the flow direction: tanθ = 4.0/3.0 = 4/3 ⇒ θ = 53°.
3.11 — Rain–man: walk 3.0 km/h, rain falls 4.0 km/h vertically
vrain,man = vrain,street − vman,street (flip the man's velocity and add).
vrain,man = √(4.0² + 3.0²) = 5.0 km/h (again a 3-4-5 triangle).
Angle with the vertical: tanθ = 3.0/4.0 = 3/4 ⇒ θ = 37°. The man tilts his umbrella 37° forward.
Bonus — Relative acceleration between two frames
Differentiating the relative-velocity relation: aP,S = aP,S′ + aS′,S.
If frame S′ moves with uniform velocity relative to S, then aS′,S = 0, so aP,S = aP,S′ — the acceleration of a body is the same in both frames.
Relative acceleration appears only when a frame is itself accelerating (a non-inertial frame).
