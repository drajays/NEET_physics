Significant Digits

Counting, rounding and combining measurements with the correct number of significant figures — the rules and worked logic from Chapter 2 (HC Verma), plus an unlimited NEET-UG practice quiz covering every scenario.

The rules at a glance

Counting significant figures

  • All non-zero digits are significant. 234 → 3
  • Zeros between non-zero digits count. 2.03 → 3
  • Leading zeros are not significant. 0.0023 → 2
  • Trailing zeros after a decimal are significant. 6.00 → 3, 100.0 → 4
  • Trailing zeros in a bare integer are ambiguous — use powers of 10. 600 → write 6×10², 6.0×10² or 6.00×10²

Rounding off

  • Next digit > 5 → round up.
  • Next digit < 5 → leave unchanged.
  • Next digit exactly 5 (nothing after) → make the kept digit even: up if it is odd, unchanged if even.
  • 14.745→14.7, 14.750→14.8, 14.650→14.6

Multiplication & division

  • Answer keeps as many significant figures as the quantity with the fewest.
  • 12.0 / 7.0 = 1.7 (2 s.f.)
  • 25.2 × 1374 / 33.3 = 1040 (3 s.f.)

Addition & subtraction

  • Round every number to the least number of decimal places present, then add/subtract.
  • 24.36 + 0.0623 + 256.2 → 24.4 + 0.1 + 256.2 = 280.7

Least count

  • The smallest sub-division of the scale.
  • A mm-marked metre scale → least count = 1 mm = 0.1 cm.
  • The reading carries all certain digits + one doubtful digit. 10.46 cm → 4 s.f.

Errors & standard deviation

  • Best value = mean x̄ of N repeated readings.
  • σ = √[ (1/N) Σ(xᵢ − x̄)² ]; report as x̄ ± σ.
  • f = (25.4 ± 0.2) cm. ~95% chance the true value lies within x̄ ± 1.96σ.
